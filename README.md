#  SQL Sales Data Analysis Project

##  Project Overview

This project focuses on analyzing a retail sales dataset using SQL to extract meaningful business insights. The dataset includes information about orders, customers, shipping, and sales performance.

---

##  Dataset Features

* Order Details (Order ID, Order Date, Ship Date)
* Customer Information (Customer ID, Name, Segment)
* Location Data (Country, City, State, Region)
* Product Details (Category, Sub-Category, Product Name)
* Sales Metrics

---

##  Tools & Skills Used

* SQL (MySQL)
* Data Cleaning
* Aggregation Functions
* Window Functions (RANK)
* Business Data Analysis

---

##  Key Business Questions Solved

###  Sales Analysis

* Total sales generated
* Monthly and yearly sales trends
* Highest sales month

###  Shipping Analysis

* Average shipping time
* Fastest shipping mode
* Most frequently used ship mode

###  Product Analysis

* Top 5 products by sales
* Best-performing category
* Underperforming sub-categories

###  Regional Analysis

* Highest revenue region
* Top-performing states and cities

###  Customer Analysis

* Total unique customers
* Top 5 customers by sales
* Most frequent customers
* Segment-wise contribution

###  Time-Based Insights

* Day of the week with highest sales

---

##  Key Insights

* Certain ship modes significantly reduce delivery time
* A small group of customers contributes a large portion of revenue
* Sales are concentrated in specific regions and categories
* Some sub-categories consistently underperform

---

## Project Structure

* `schema.sql` → Table creation
* `analysis.sql` → All analytical queries
* `dataset.csv` → Raw data

---

## Conclusion

This project demonstrates how SQL can be used to analyze business data and generate actionable insights. It reflects strong foundational skills required for a Data Analyst role.
